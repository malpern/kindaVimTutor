## What It Does
Type `f` followed by a character to jump forward to that character on the current line. `F` searches backward. Type `t` or `T` when you want to stop just before the target instead of landing on it.

## How To Use It In KindaVim
These commands are strongest when you can already see the target on the current line. They let you jump directly to punctuation, delimiters, or a specific letter without tapping `l` ten times. After one `f` or `t`, use `;` to repeat in the same direction and `,` to reverse it.

`t` is especially useful before operators. A shape like `dt)` deletes everything up to the closing paren without deleting the paren itself.

## Difference From Stock Vim
The mental model is the same as Vim, but the scope is KindaVim's supported text surfaces. There is no full-screen editor buffer, so the value here is precise line-local navigation inside native macOS controls.
